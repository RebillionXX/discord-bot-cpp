# discord-bot-cpp
a discord bot written in c++

## How to build the source - starting path (discord-bot-cpp/)
```shell
$ mkdir build
$ cd build
$ cmake ..
$ cmake --build .
```

## Running the program
- copy a executable file (`discord_bot.exe`) from `discord-bot-cpp/build/src/debug` to `discord-bot-cpp/bin`
- run the program.
